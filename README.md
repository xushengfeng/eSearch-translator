# eSearch-translator

翻译，将适用于[eSearch](https://github.com/xushengfeng/eSearch/)翻译结果的处理。

允许使用树形模式组合翻译器。如同时调用多个翻译器，某个翻译器翻译的文字传入下一个引擎……

翻译器允许自定义引擎、语言。

引擎支持：

-   [x] 百度
-   [x] 有道
-   [x] 必应
-   [x] 彩云
-   [ ] 谷歌
-   [ ] Yandex
-   [ ] 离线

url search 参数
| 参数 | 描述 |
| ---- | ---- |
| text | 文本 |
| m | 模式 |
